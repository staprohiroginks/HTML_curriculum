色と大きさの変更

今回から新しくCSS（シーエスエス）ファイルを使います。
CSSファイルを活用して、背景色、文字の色、そして文字の大きさを変更しましょう！

CSSとは？

CSSとは "Cascading Style Sheets" の略で、主にホームページの見た目についてプログラミングする言語です。
「Webページ = HTML（構造的な文書）+ CSS（見た目）」と理解しておきましょう。
見た目を変更したいときは、以下の書式に従う必要があります。 

セレクタ {
    プロパティ: 値;
}

  • セレクタ：スタイルを適用する対象を指定します。 <body>に適用したいなら「body」、<p>に適用したいなら「p」が入ります。

  • プロパティ：どんな装飾をするかを指定します。 色を変更したいなら「color」、文字の大きさを変更したいなら「font-size」が入ります。

  • 値：プロパティに対する値を指定します。色の名前や、文字の大きさを表す数値（px）が入ります。

CSSファイルの作成・読み込み

まずは新しくCSSファイルを作りましょう。作る場所は、今コードを書いているindex.htmlと必ず同じ場所（同じ階層）にしてください。
Visual Studio Codeのマークを押して、「style.css」という名前を付けます。
この拡張子「.css」がCSSファイルの目印です。

次に、HTMLファイルの中でCSSファイルを読み込みます。
HTMLファイルを開き、ヘッダータグ<header>の中に以下の一文を書きます。 

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="utf8">
    <meta name="description" content="初めてHTMLを勉強するサイトです。">
    <link rel="stylesheet" href="style.css"> <!-- CSSファイルの読み込み --> 
    <title>プログラミングの勉強</title>
</head>

これでCSSファイルをHTMLファイルとつなげることができました！
なお、<!-- -->でコンテンツを囲むことをコメントアウトといいます。
囲んだ部分は非表示にすることができるため、一時的に消去したりコードの内容の補足を書いたりする場合に使えます。
Macでは Cmd + / キー、Windowsでは Ctrl + / キーで選択部分をコメントアウトできます。

色の変更

背景と文字の色を変更しましょう！
背景色は<body>を指定し、文字の色は<p>を指定して変更します。
また、背景は英語で「background」と訳されます。
例として背景色を黄色に、文字の色を赤色にしてみましょう。

HTMLファイル

<body>
    <p>こんにちは！</p>
</body>

style.css

body {
    background-color: yellow;
}
p {
    color: red;
}

実行例

文字の大きさの変更

色と同じように、文字の大きさも変更しましょう！
pタグはデフォルトで16pxになっています。
先ほどのCSSファイルに1行追加します。

HTMLファイル

<body>
    <p>こんにちは！</p>
</body>

style.css

body {
    background-color: yellow;
}
p {
    color: red;
    font-size: 30px;
}

実行例

予備知識

色の指定方法

色のプロパティ「color: 〇〇;」に入る値は、合わせて4通りの指定方法があります。
状況に応じて使い分けるのがおすすめです。

指定形式 値の指定例 概要
キーワード 「red」「blue」「green」
  • 17種類の色の名前で指定
  • すべてのブラウザが対応している上に分かりやすい
16進数カラーコード 「#ffcc00」「#c00」
  • 「# + カラーコード」で指定
  • デザイン制作では主流の指定方法
RGB 「rgb( 255, 42, 53)」
  • 光の3原色であるRed、Green、Blueのカラーコードを指定
  • 透過度A(Alpha)も指定するなら「rgba( 255, 0, 0, 0.5)」と表記
HSL 「hsl( 0, 100%, 100%)」
  • Hue(色相)、Saturation(彩度)、Lightness(輝度)の3つを指定
  • 透過度A(Alpha)も指定するなら「hsla( 207, 100%, 86%, 0.5);」と表記

文字の見た目

文字の見た目は、色や大きさ以外にも様々な変更を加えることができます。
以下はその一覧です。

プロパティ 値の指定例 変更する部分
font-size 「10px」「2.0em」 大きさ（emは初期設定値に対する倍率）
color 「red」「#ffcc00」 色
font-weight 「bold」「normal」「lighter」 太さ
font-size 「normal」「italic」「oblique」 傾き（italic：筆記体、oblique：斜体）
text-decoration 「underline」「line-through」 下線、取り消し線
text-align 「left」「center」「right」 位置（初期値はleft）

これらのプロパティはかなり頻繁に使用するため、しっかり覚えましょう！
