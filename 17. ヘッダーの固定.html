ヘッダーの固定

15. header、main、footerについて.htmlで学習したheaderを固定して、ヘッダーが常に表示されるようにしましょう。

ヘッダーの固定によるタイトル表示

まずは以下のコードを用意しましょう。

index.html

<header>

    <nav>
        -ホーム
        -家庭用ゲーム機
        -スマートフォン
        -PC
        -その他
    <nav>
</header>

<main>
    <p>テキスト1</p>
    <p>テキスト2</p>
    <p>テキスト3</p>
    <p>テキスト4</p>
    <p>テキスト5</p>
    <p>テキスト6</p>
    <p>テキスト7</p>
    <p>テキスト8</p>
    <p>テキスト9</p>
    <p>テキスト10</p>
    <p>テキスト12</p>
    <p>テキスト13</p>
    <p>テキスト14</p>
    <p>テキスト15</p>
    <p>テキスト16</p>
    <p>テキスト17</p>
    <p>テキスト18</p>
    <p>テキスト19</p>
    <p>テキスト20</p>

<footer>

style.css

body {
    font-size: 200%;
}
header {
    background-color: skyblue;
    text-align: center;
}

実行結果

このままではヘッダーが固定されず、見切れてしまいます。
固定するためには、positionプロパティでfixedを指定します。
なお、fixは英語で「固定する」という意味です。
cssファイルのh1を以下のように書き換えましょう。

style.css

header {
    background-color: skyblue;
    text-align: center;
    position: fixed;
    width: 100%;
    top: 0px;
    left: 0px;
}

実行結果

これでヘッダーが固定されるようになりました。
しかし、このままではテキスト1~3がヘッダーに被ってしまっています。
そこで、カリキュラム12. margin、padding、borderを復習しながら、
ページ内容全体の上に、ヘッダーの高さと同じ長さの余白を入れましょう。
cssファイルに以下のコードを追加します。

style.css

main {
    margin-top: 250px;
}

実行結果

ヘッダーと画像の重なりが解消され、正しく表示されるようになりました！

ページ上部への復帰

次は画面がスクロールしても常にトップへ戻れるようにしてみましょう。
画面右下に「トップへ戻る」ボタンを配置し、常に表示されるように固定します。

headerやmain、footerとは独立した要素に、ページトップへ戻るリンク"#"を設定します。
リンクでhref属性に"#"を設定する場合を以下に示します。

  • ページのトップに戻るため。
  • リンクを仮設定し、後から別のリンクに書き換えるため。
  • 同じページ内で、id属性を設定した特定の場所にジャンプさせるため。

htmlファイルのfooterの直下とcssファイルに、以下のコードを追加しましょう。

index.html

<div class="top">
  <a href="#">トップへ戻る</a>
<div>

style.css

.top {
    position: fixed;
    bottom: 10px;
    right: 20px;
    width: 200px;
    height: 50px;
    background-color: skyblue;
}

実行結果

ボタンを押して、ページ上部に復帰出来るようになりました！
