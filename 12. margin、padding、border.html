margin、padding、border

margin（マージン）とpadding（パディング）は、直訳するとそれぞれ「空白」と「詰め物」という意味です。
HTMLではそれぞれ「外側の空白」と「内側の空白」に該当します。
また、これらを区切る線をborder（ボーダー）で定義できます。

外側の空白：margin

まずは以下のコードを用意しましょう。

index.html

<div class="stapro1">
    <p>スタートプログラミング1</p>
</div>
<div class="stapro2">
    <p>スタートプログラミング2</p>
</div>

style.css

body {
    background-color: skyblue;
}
.stapro1,.stapro2 {
    background-color: pink;
    width: 250px;
    height: 50px;
}

実行結果

続いて、cssファイルに対してmarginに関する1文加えてみましょう。

style.css

body {
    background-color: skyblue;
}
.stapro1,.stapro2 {
    background-color: pink;
    width: 250px;
    height: 50px;
    margin: 30px;
}

実行結果

それぞれのピンク色のエリアの周り（上下左右）に水色の部分が増えました！
2つのエリアの間隔も広くなったのが分かります。
このように、marginは枠の外側に対して空白を作ることができます。
なお値を「-30px」とすることで、空白を狭くすることができます。

内側の空白：padding

次は、内側に空白を作ります。
先ほどのcssファイルに対して、paddingに関する1文加えてみましょう。

style.css

body {
    background-color: skyblue;
}
.stapro1,.stapro2 {
    background-color: pink;
    width: 250px;
    height: 50px;
    margin: 30px;
    padding: 10px;
}

実行結果

ピンク色のエリアが上下左右に広がりました！
このように、paddingは枠の内側に対して空白を作ることができます。

空白の設定方法の種類

空白のプロパティであるmarginとpaddingは、今まで上下左右すべて一括で指定してきました。
他にも様々な指定方法があります。

marginと値の種類 paddingと値の種類 意味
margin:10px; padding:10px;

上下左右すべてに10px
margin:10px 20px; padding:10px 20px;

上下に10px、左右に20px
margin:10px 20px 30px; padding:10px 20px 30px;

上に10px、左右に20px、下に30px
margin:10px 20px 30px 40px; padding:10px 20px 30px 40px;

上に10px、右に20px、下に30px、左に40px
margin-top:10px; padding-top:10px;

上のみ10px
margin-bottom:10px; padding-bottom:10px;

下のみ10px
margin-left:10px; padding-left:10px;

左のみ10px
margin-right:10px; padding-right:10px;

右のみ10px
margin:0 auto; ---

空白無しで自動中央揃え

最下行の自動中央揃えはmarginのみの指定方法です。

線：border

次は、枠に線を加えます。
線のプロパティであるborderは、cssファイルで以下のように指定します。 

セレクタ{
    border: 線のスタイル 線の太さ 線の色;
}

では先ほどのcssファイルに対して、borderに関する1文を加えてみましょう。

style.css

body {
    background-color: skyblue;
}
.stapro1,.stapro2 {
    background-color: pink;
    width: 250px;
    height: 50px;
    margin: 30px;
    padding: 10px;
    border: solid 2px black;
}

実行結果

枠に黒線が表れました！
このように、borderは枠の線を指定することができます。
特にsolidは「実線」を指定する値であるため、dashedにすると「点線」を指定できます。

style.css

body {
        background-color: skyblue;
    }
    .stapro1,.stapro2 {
        background-color: pink;
        width: 250px;
        height: 50px;
        margin: 30px;
        padding: 10px;
        border: dashed 2px white;
    }

実行結果
