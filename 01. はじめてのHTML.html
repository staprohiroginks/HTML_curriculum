はじめてのHTML

このカリキュラムでは、HTMLというプログラミング言語で簡単なホームページを作っていきます。 HTMLの知識を増やし、最終的には自己紹介のホームページを作成し、検定に挑戦してもらいます！

HTMLの準備

まずはHTMLを学ぶ環境を整えましょう。

  1. 教材ダウンロードページからHTML前半コースの教材をダウンロードします。
    （個人のパソコンを持っている生徒はダウンロードした教材をデスクトップに張り付けることをおすすめします。）
  2. Visual Studio Codeを開いて、右上の「ファイル(file)」→「フォルダを開く(open folder)」でダウンロードした教材を選択します。
  3. フォルダを開いたら、右上の「ファイル(file)」→「新規ファイル(new file)」で新しいファイルを作ります。
  4. ファイル名を"index.html"にして、ファイルを保存（Command + S または Ctrl + S）します。

コードのテンプレート

HTMLでコードを書いていくためには、最初におまじないのようなコードを書く必要があります。
コードの全ての意味を深く覚える必要はないですが、おまじないのコードを出す方法は覚えましょう！ 　　　　　　　　　　　　　　　　　　　　↓
HTMLファイルを作ったら、小文字で「html」と入力します。
すると、予測変換で「html:5」というのが出てくるので、それをクリックしましょう。
そうすると、HTMLを書いていくために必要なコードが出てきます。

Webページの表示

簡単な文字をWebページに表示してみましょう！
まずはVisual Studio Codeに文字を打ち込みます。

例

<body>
    Hello World!
</body>

次にこのindex.htmlファイルをGoogle Chrome（グーグルクローム）で開きます。 先ほど打ち込んだ文字がWebページに表示されます。

実行結果

Hello World!

用語とショートカットの解説

以下はHTMLを学ぶにあたりよく出てくる用語です。

  • ブラウザ：ホームページを見るもの
  • エディタ：プログラムをタイピングしていくメモ帳
  • ソースコード：プログラミングした文章のこと
  • 拡張子：ファイルを判断する名前（ ".(ドット)"の後ろ部分 ）
  • HTMLソース：HTML言語で書かれたソースコード（.html）
  • CSSソース：CSS言語で書かれたソースコード（.css）
今回はブラウザをGoogle Chrome、エディタをVisual Stdio Codeとしています。
また、以下はキーボードの便利なショートカットです。覚えておきましょう！
  • 保存　　　　　： (Mac)Command + S 　　(Windows) Ctrl + S
  • コピー　　　　： (Mac)Command + C 　　(Windows) Ctrl + C
  • 切り取り　　　： (Mac)Command + X 　　(Windows) Ctrl + X
  • 貼り付け　　　： (Mac)Command + V 　　(Windows) Ctrl + V
  • コメントアウト： (Mac)Command + / 　　(Windows) Ctrl + /
  • ページ更新　　： (Mac)Command + R 　　(Windows) Ctrl + R
