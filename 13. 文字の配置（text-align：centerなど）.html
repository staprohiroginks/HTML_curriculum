文字の配置（text-align：centerなど）

CSSファイルで、文字の位置や表示方法について変更することが出来ます。

文字のプロパティ一覧

文字のプロパティは、一例として以下が挙げられます。 

p {
    // フォントサイズ（サイズ14ピクセル）
    font-size: 14px;
    // フォントの太さ（太字）
    font-weight: bold;
    // フォントカラー（青）
    color: blue;
    // フォントの透明度
    opacity: 0.5;
    // フォントのスタイル（斜体）
    font-style: italic;
    // 文字を揃える方向（中央）
    text-align: center;
    // 文字の装飾（下線）
    text-decoration: underline;
    // 文字の行間（サイズ10ピクセル）
    line-height: 10px;
}

フォントサイズと文字の行間は数値で、フォントカラーはカリキュラム7で学んだ方法で指定します。
それ以外のプロパティは以下で説明します。

フォントの太さ

フォントの太さを指定するプロパティfont-weightの値は、以下が挙げられます。

プロパティの値
font-weight:〇〇;		 効果 表示例
normal 普通

フォントの太さ
lighter やや細い

フォントの太さ
bolder やや太い

フォントの太さ
bold 太い

フォントの太さ
100 カスタム

フォントの太さ

現在表示しているフォントではあまり変化がありません。
フォントの種類によって大きく変化が現れるものもあります。

index.html

<p class="before">css変更前</p>
<p class="after">css変更後</p>

style.css

body {
    font-size: 30px;
}
.after {
    font-weight: bold;
}

文字を揃える方向を指定するプロパティtext-alignの値は、以下が挙げられます。

プロパティの値
text-align:〇〇;		 効果 表示例
left 左寄せ

文字を揃える方向
right 右寄せ

文字を揃える方向
start 開始方向寄せ

leftと同義

※日本語・英語は
rightと同義

文字を揃える方向
center 中央寄せ

文字を揃える方向
justify 最終行除く両端揃え

文字を揃える方向はtext-alignで指定します。

style.css

.after {
    text-align: center;
}

文字の装飾

文字の装飾を指定するプロパティtext-decorationの値は、以下が挙げられます。

プロパティの値
text-decoration:〇〇;		 効果 表示例
none 線の表示なし

文字の装飾
underline 下線

文字の装飾

文字の装飾

underlineとoverlineは同時に指定することができます。
また、文字の装飾は「プロパティ : 線のスタイル 線の色 値 ;」と指定することもできます。

style.css

.after {
    text-decoration: dashed red underline;
}
