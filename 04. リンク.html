リンク

リンクの設定をしてみよう！

リンクをつくる

リンクを貼るためには<a>タグを使います。 今回は、Yahooのページのリンクを作ってみます。

例をお手本にhref属性にURLを書きましょう！

例

<a href="https://yahoo.co.jp">yahoo</a>
  • URL(ユーアールエル)：ホームページの住所のようなもの。サイトごとにURLが決まっています。

実行結果

yahoo

これでYahooのページに行くことができました！


新しいタブでひらく

リンクを押すと、今あるページがリンク先のページに変わってしまいます。
今のページはそのままに、リンク先を新しいタブで開けるようにしましょう！

URLの後ろにtarget="_blank" と入れることで新しいタブでサイトを開けるようになります。

例

<a href="https://yahoo.co.jp" target="blank">yahoo</a>

実行結果

yahoo

これで、新しいタブで開くことができました！


作った別のHTMLファイルに移動しよう

これまでindex.htmlファイルしか作ってきませんでしたが、複数のページを作ることもできます。

今回は新しく、test.htmlという名前のファイルを作って、そこへリンクで飛べるようにしてみましょう。
※index.htmlとtest.htmlは同じ階層に作ってください

新しく作ったtest.htmlの内容は、以下の内容にしましょう。そのままコピーして貼り付けてください。

test.html

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="utf8">
    <meta name="description" content="初めてHTMLを勉強するサイトです。">
    <title>test.html | リンクの使いかた</title>
</head>
<body>
    <p>
       index.htmlから、このページへ飛べたら成功！
    </p>
</body>
</html>

では、index.html の方にもどって、test.html のリンクをはってみよう！

例（index.html）

<a href="test.html">テストページ</a>

これで index.html から test.html へ移動することできました！
