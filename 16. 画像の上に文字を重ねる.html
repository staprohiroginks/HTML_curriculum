画像の上に文字を重ねる

このように画像の上に文字を重ねたい場合、positionプロパティとz-indexプロパティを使います。

positionプロパティ

positionプロパティは、配置する方法を指定します。
配置の指定方法は「relative（相対位置）」と「absolute（絶対位置）」の2種類があります。
positionプロパティを使用したら、加えてtop、left、right、およびbottomなどのプロパティで
配置する距離の長さを指定します。
まずは画像と文字を準備しましょう。

index.html

<body>
    <div class="box">
        <div class="image">
            <img src="cat.jpg" alt="cat">
        </div>

        <div class="text">
            <p>ここにテキストを記入します。</p>
        </div>
    </div>
</body>

style.css

.text {
    width: 250px;
    height: 50px;
    border: solid 2px red;
}

実行結果

続いて画像と文字をまとめて移動させましょう。
このとき、親要素であるboxクラスの位置をrelative（相対位置）で指定します。
boxクラスに関して、以下のコードを新たに書き加えます。

style.css

.box {
    position: relative;
    top: 50px;
    left: 100px;
    width: 520px;
}
img {
    width: 100%;
}

実行結果

画像の位置を指定することができました！
この場合『本来表示される箇所を基準として、上から50px、左から100px離れた位置』と指定しています。
次は親要素であるboxクラスの中で、子要素である文字の位置をabsolute（絶対位置）で指定します。
textクラスに関して、以下のコードに書き換えます。

style.css

.text {
    width: 250px;
    height: 50px;
    border: solid 2px red;
    position: absolute;
    top: 150px;
    left: 50px;
}

実行結果

文字の位置を指定することができました！
この場合『親要素（今回だとbox）の左上を基準として、上から150px、左から50px離れた位置』と指定しています。
親要素の右や下を基準にすることもできます。

style.css

.text {
    width: 250px;
    height: 50px;
    border: solid 2px red;
    position: absolute;
    bottom: 20px;
    right: 0px;
}

実行結果

この場合『親要素の右下を基準として、下から20px、右から0px離れた位置』と指定しています。

z-indexプロパティ

z-indexプロパティは重なり方を指定します。
値は正の整数または負の整数で指定することができ、数値が大きいものをより手前に表示します。
まずは下記のように3つのボックスをabsolute（絶対配置）で重ねてみます。

index.html

<body>
    <div class="box1">
        <p>ボックス1</p>
    </div>
    <div class="box2">
        <p>ボックス2</p>
    </div>
    <div class="box3">
        <p>ボックス3</p>
    </div>
</body>

style.css

.box1 {
    background-color: #faa3a3;
    width: 200px;
    height: 200px;
    position: absolute;
}
.box2 {
    background-color: #afc9f1;
    width: 200px;
    height: 200px;
    position: absolute;
    top: 50px;
    left: 50px;
}
.box3 {
    background-color: #bcf3b1;
    width: 200px;
    height: 200px;
    position: absolute;
    top: 100px;
    left: 100px;
}

実行結果

次に重なり順を変更してみます。
先程は緑のboxが一番上、青のboxが真ん中、赤のboxが一番下にありましたが、その重なり順をz-indexを使って逆にしてみます。
それぞれのクラスに以下のコードを追加しましょう。

style.css

.box1 {
    z-index: 3;
}
.box2 {
    z-index: 2;
}
.box3 {
    z-index: 1;
}

実行結果

このようにz-indexを上手く使うことで、要素の重なり順を変更することが出来ます！

背景画像の差し込み

画像を入れる方法として、htmlファイルから読み込む方法とcssファイルから読み込む方法があります。
今まではHTMLファイルにimgタグで画像を指定していましたが、今度はCSSファイルにURLで指定します。
今回はcssフォルダとimgフォルダを作成し、相対パスを用いて指定しましょう。
まずは以下のように、2つのフォルダにそれぞれcssファイルと画像を入れます。

次に、HTMLファイルとcssファイルのコードを以下のように書き換えましょう。
cssファイルではbackground-imageプロパティでURLを指定しています。
今回のURL指定では、imgフォルダを指定する前に「カリキュラム16のフォルダに戻る」作業をします。
つまり階層を1つ上に移動するため、URLの先頭に"../"が付きます。

index.html

<head>
    ---省略---
    <link rel="stylesheet" href="css/style.css"> 
</head>
<body>
    <p>ここにテキストを記入します。<p>
</body>

style.css

body {
    background-image: url(../img/cat.jpg);
}
p {
    color: red;
}

実行例

背景画像が文章の裏側に表示されました！
初期状態では、背景画像が範囲いっぱいに繰り返し表示されるようになっています。
次は、繰り返し画像が表示されないようにするため，background-repeatプロパティを追加します。

style.css

body {
    background-image: url(../img/cat.jpg);
    background-repeat: no-repeat;
}

実行例

背景画像が1枚のみになりました！
背景画像の繰り返しは、このほかに以下のような指定方法があります。

body {
