テーブル

データを見やすく整理するための表を作成しましょう！

表の構成

表のことをテーブル（table）といいます。 右図はHTMLで表を作るときの構造です。

表を作るときは行ごとに内容を書いていきます。
表の中のひとつひとつのマスをセルと言い、項目の名前を見出しセル、項目ごとのデータが入るセルをデータセルと言います。
それぞれの英訳とタグは以下のとおりです。

テーブルの行　　 → table row　　 → <tr>
テーブルの見出し → table header　→ <th>
テーブルのデータ → table data　　→ <td>

今回使用するタグは<table>、 <caption>、 <th>、 <tr>、 そして<td>です。

データセルの表示

まずは<table>、 <tr>、 そして<td>を使って、データセルだけを表示してみましょう。

例

<table>
    <tr>
        <td>りんご</td>
        <td>200円</td>
    </tr>
    <tr>
        <td>ぶどう</td>
        <td>150円</td>
    </tr>
    <tr>
        <td>みかん</td>
        <td>50円</td>
    </tr>
</table>

実行結果

りんご 200円
ぶどう 150円
みかん 50円

このままだと線が表示されません。
線を表示するためにborder="1"を追加しましょう。
border（ボーダー）は英語で線という意味です。
border="1"は1本線、 border="2"は2本線となり、 border="0"の場合は線を表示しないことになります。

例

<table border="1">
    <tr>
        <td>りんご</td>
        <td>200円</td>
    </tr>
    <tr>
        <td>ぶどう</td>
        <td>150円</td>
    </tr>
    <tr>
        <td>みかん</td>
        <td>50円</td>
    </tr>
</table>

実行結果

りんご 200円
ぶどう 150円
みかん 50円

1本線にするつもりが2本線になってしまいました。 この原因は、今は「一つ一つの枠を1本線で囲む」という状態になっており、隣り合う線を打ち消しあっていないからです。
(テーブル要素の初期設定でstyle="border-collapse: separate;"という命令が入っており、これはセルの境界線が個別に表示されるという意味になっています)

そのため、線を打ち消しあうためにstyle="border-collapse: collapse;"を追加しましょう。
この命令を入れることで、セルの境界線が一つにまとまり、隣接するセル間で境界線が共有されます。

例

<table border="1" style="border-collapse: collapse;">
    <tr>
        <td>りんご</td>
        <td>200円</td>
    </tr>
    <tr>
        <td>ぶどう</td>
        <td>150円</td>
    </tr>
    <tr>
        <td>みかん</td>
        <td>50円</td>
    </tr>
</table>

実行結果

りんご 200円
ぶどう 150円
みかん 50円

タイトルと見出しセル

次にタイトルと見出しセルを追加しましょう。
表のタイトルはキャプションと言い、タグは<caption>を使います。
テーブルの見出しのタグは<th>です。見出しは文字が太文字になります。

例

<table border="1" style="border-collapse: collapse;">
    <caption>買い物リスト</caption>
    <tr>
        <th>果物</th>
        <th>値段</th>
    </tr>
    <tr>
        <td>りんご</td>
        <td>200円</td>
    </tr>
    <tr>
        <td>ぶどう</td>
        <td>150円</td>
    </tr>
    <tr>
        <td>みかん</td>
        <td>50円</td>
    </tr>
</table>

実行例

買い物リスト
果物 値段
りんご 200円
ぶどう 150円
みかん 50円

セルの結合

先ほどのテーブルにセルを追加して、さらにセル同士を結合してみましょう。
例えば、縦2つ分のセルを結合したいときは、一番上のセルのタグ<td>を<td rowspan="2">に変更します。
また、横2つ分のセルを結合したいときは、一番左のセルのタグ<td>を<td colspan="2">に変更します。

例

<table border="1" style="border-collapse: collapse;">
        <caption>買い物リスト</caption>
        <tr>
            <th>果物</td>
            <th>値段</td>
        </tr>
        <tr>
            <td>りんご</td>
            <td>200円</td>
        </tr>
        <tr>
            <td>ぶどう</td>
            <td>150円</td>
        </tr>
        <tr>
            <td rowspan="2">みかん</td>
            <td>50円</td>
        </tr>
        <tr>
            <td>100円</td>
        </tr>
        <tr>
            <td colspan="2">※みかんは2種類買う予定</td>
        </tr>
    </table>

実行例

買い物リスト
果物 値段
りんご 200円
ぶどう 150円
みかん 50円
100円
※みかんは2種類買う予定

予備知識

見出しの位置（scope）

この属性は、th要素に対して使用され、そのヘッダーセルがどの範囲にデータセルを関連付けているかを指定します。
見た目は変わりませんが、データセルが見出しセルに紐づけられている印になります。
例えば、同じ行（水平方向）のデータセルを紐づけるときは、タグ<th>を<th scope="row">に変更します。
また、同じ列（垂直方向）のデータセルを紐づけるときは、タグ<th>を<th scope="col">に変更します。
