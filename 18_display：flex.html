display：flex

このカリキュラムではWebページのレイアウトを変更するdisplayプロパティを学習します。

display: flex;（要素の横並び）

displayプロパティは要素の表示形式を定義するプロパティです。
値をflexに指定することで、要素を横並びにすることができます。
一つ一つの要素（子要素）に対して同じ名前のclassを指定し、
さらに横並びにしたい要素全体（親要素）に別名のclassを付けます。
flexを用いると、以下の3つのメリットがあります。

  1. 高さを自動で揃えてくれる
  2. 簡単に並び順を指定できる
  3. 簡単に余白の取り方を指定できる

まずは以下のコードを書きましょう。

index.html

<div class="box">
  <div class="item">
      <p>HTML前半コース最後のカリキュラムです。頑張りましょう！</p>
  </div>
  <div class="item">
      <p>HTML前半コース最後のカリキュラムです。頑張りましょう！</p>
  </div>
  <div class="item">
      <p>HTML前半コース最後のカリキュラムです。頑張りましょう！</p>
  </div>
  <div class="item">
      <p>HTML前半コース最後のカリキュラムです。頑張りましょう！</p>
  </div>
</div>

style.css

.box {
    background-color: black;
    padding: 10px;
    }
.item {
    margin: 10px;
    padding: 10px;
}
.item:nth-child(1) {
    background-color: pink;
}
.item:nth-child(2) {
    background-color: yellow;
}
.item:nth-child(3) {
    background-color: skyblue;
}
.item:nth-child(4) {
    background-color: lightgreen;
}

実行結果

これらの要素を横並びにするために、displayプロパティでflexを指定します。
cssファイルのboxを以下のように書き換えましょう。

style.css

.box {
    background-color: black;
    padding: 10px;
    display: flex;
}

実行結果

横並びにすることができました！
幅を指定していないため、ウィンドウの大きさに合わせて均等に幅が変化します。

justify-contentプロパティ

横並びの具体的な方法はjustify-contentプロパティで指定します。
基本的に、displayプロパティでflexを指定したあとに使います。
幅を指定する場合、初期状態は左寄せになっています。
cssファイルのitemを以下のように書き換えて確かめましょう。

style.css

.item {
    margin: 10px;
    padding: 10px;
    width: 100px;
}

実行結果

初期状態が左寄せであることが分かりました。
ここから、cssファイルのboxを以下のように書き換えて、中央寄せにしてみましょう。

style.css

.box {
    background-color: black;
    padding: 10px;
    display: flex;
    justify-content: center;
}

実行結果

中央寄せにすることができました！
そのほかに以下のような指定方法があります。

値の種類
justify-content:〇〇;		 効果 例
left

左寄せ
right

右寄せ
center

中央寄せ
space-between

最初と最後を
端に配置
space-around

均等揃え

displayの値の種類

displayプロパティの値はflexのほかに以下のような値があります。

値の種類
display:〇〇;		 内容 初期設定されている
主なタグ
block
  • インライン要素を「ブロック要素」に変化させる
  • 外枠と同じ様に扱う
  • 兄弟要素を縦に配置する
  • widthを指定できる（しなければ横幅いっぱいになる）
<div>
<h1>
<ol>
<ul>
<blockquote>
inline
  • ブロック要素を「インライン要素」に変化させる
  • テキストと同じ様に扱う
  • 兄弟要素を横に配置する
  • widthを指定できない
<a>
<b>
<br>
<img>
<input>
<small>
<strong>
inline-block
  • 外枠と同じ様に扱う
  • 兄弟要素を縦に配置する
  • widthを指定できる
flex
  • 兄弟要素を横に配置する
  • レイアウト調整に使われる
none
  • 指定した部分を全て消す

例として、まずはブロック要素であるh1タグを並べてみましょう。

index.html

<body>
    <h1>HTML前半コース最後のカリキュラムです。頑張りましょう！</h1>
    <h1>HTML前半コース最後のカリキュラムです。頑張りましょう！</h1>
    <h1>HTML前半コース最後のカリキュラムです。頑張りましょう！</h1>
</body>

実行結果


この場合、以下のようにh1タグが横いっぱいに広がっています。

これをインライン要素に置き換えてみましょう。

style.css

h1 {
    display: inline;
}

実行結果

インライン要素にすることで、h1タグが以下のように切り詰められます。

次に、インライン要素であるimgタグの画像を並べてみましょう。

index.html

<body>
    <img src="cat.jpg" alt="cat" width="100px">
    <img src="cat.jpg" alt="cat" width="100px">
    <img src="cat.jpg" alt="cat" width="100px">
</body>

実行結果


この場合、以下のようにimgタグが横並びに詰められています。
これをブロック要素に置き換えてみましょう。

style.css

img {
    display: block;
}

実行結果

ブロック要素にすることで、imgタグごとに改行されました。

float（回り込み）

floatとは英語で「浮く」という意味です。
CSSファイルにおいてプロパティfloatとは要素の回り込みを設定し、レイアウトを変更するものです。
例えば、ニュース記事の横に画像を挟みたい場合、画像を文章の横に回り込ませることで、
新聞のように文字を無駄なく配置することができます。
まずは以下のコードを書きましょう。

index.html

<body>
    <div class="pot">
        <h1>ねこ鍋とは</h1>
        <img src="cat.jpg" alt="cat" width="200px">
        <p>        
            ねこ鍋とは、猫が土鍋の中で身を丸くする現象である。
            猫は狭い場所を好むとされ、この現象も猫の習性の一つといえる。
            一般的には、各々が自分に合った大きさの土鍋を見つけて収まっていく。
            既に他の子猫が寝ている土鍋に割り込んだり折り重なったり、
            寝返りを打った際に土鍋から転げ出したりすることもある。
            土鍋への猫の入る匹数によって、1匹入るのを「並盛り」、2匹は「大盛り」、
            3匹は「特盛り」、4匹は「激盛り」と表現される。
        </p>
    </div>
</body>

style.css

.pot {
    width: 500px;
}

実行結果


次に、画像に回り込みを設定します。
CSSファイルにおいて、以下のコードを追加しましょう。

style.css

img {
    float: left;
}

実行結果


文章の左側に画像を回り込ませることができました！
これは「文章が移動して空白が詰められた」のではなく「文章の枠に画像が割り込んだ」ということです。
同様に右側にも設定することができます。

style.css

img {
    float: right;
}

実行結果


文章の右側に画像を回り込ませることができました！
なお、回り込みの設定を適用しない場合、floatプロパティの値はnoneにします。

box-sizing: border-box;

box-sizingプロパティは要素の幅および高さの計算方法を設定するプロパティです。
値をborder-boxに設定することで、paddingやborderも含めた計算ができるようになります。
まずは先ほどのHTMLファイルおよびCSSファイルを以下のように書き換えましょう。

index.html

<body>
    <div class="pot">
        <h1>ねこ鍋とは</h1>
        <div class="content">
            <p>        
                ねこ鍋とは、猫が土鍋の中で身を丸くする現象である。
                猫は狭い場所を好むとされ、この現象も猫の習性の一つといえる。
                一般的には、各々が自分に合った大きさの土鍋を見つけて収まっていく。
                既に他の子猫が寝ている土鍋に割り込んだり折り重なったり、
                寝返りを打った際に土鍋から転げ出したりすることもある。
                土鍋への猫の入る匹数によって、1匹入るのを「並盛り」、2匹は「大盛り」、
                3匹は「特盛り」、4匹は「激盛り」と表現される。
            </p>
        </div>
    </div>
</body>

style.css

.pot {
    width: 500px;
    height: 500px;
    border: solid 5px black;
    }
.content {
    width: 100%;
    height: 300px;
    padding: 20px;
    border: solid 10px red;
}

実行結果

これでは枠の外に赤い枠線や文字がはみ出してしまいますね。
では先ほどのcssファイルに対して、box-sizingに関する1文を加えてみましょう。

style.css

.pot {
    width: 500px;
    height: 500px;
    border: solid 5px black;
    }
.content {
    box-sizing: border-box;
    width: 100%;
    height: 300px;
    padding: 20px;
    border: solid 10px red;
}

実行結果

赤枠の内容を黒枠内に収めることができました！
高さもpaddingやborderを含めて計算されています。
このように、box-sizingでborder-boxを設定することで、内側の空白や枠線も含めた要素のサイズ設定ができます。

練習問題

今まで学んできた知識を活かして、以下の画像と同じようなページを作ってみましょう。
（見本はこちら）

  • ヘッダー・メイン
    • ヘッダーは固定してください。
    • ヘッダーの高さは100pxに設定し、メインをその高さの分だけ下から開始してください。
  • 薄緑色のエリア
    • 背景色をlightgreen、幅を800px、内側の空白を10pxにしてください。
    • 1つのボックス（クラス）を中央に寄せるには、cssに「margin:0 auto;」を加えます。
    • 今回のカリキュラムの前半を参照しながら、各月のエリアを横並べにしてください。
    • 横並べの時に折り返すには、cssに「flex-wrap: wrap;」を加えます。
  • 各月のエリア
    • 背景色をwhite、幅を200px、外側の空白を10px、内側の空白を10pxにしてください。
    • 文章は、tableタグ（線の表示なし）を使うとよりきれいに整います。
