27.ボタンのレスポンシブ対応

2つのボタンがあり、画面サイズに合わせてボタンの大きさや位置を変える方法を学習しましょう！

見本はこちら

①まずはHTMLでボタンを2つ用意します。

ボタンを押すと詳細ページに飛ぶようにしたいので、今回は、左側のボタンを押すとスタプロの紹介ページ、
右側のボタンを押すとスタプロのコース紹介ページに飛ぶようにリンクを設定します。

<div class="button">
    <div class="stapro">
        <a href="https://www.start-programming.net/about/" target="_blank"><p>スタプロとは？</p></a>
    </div>

    <div class="course">
        <a href="https://www.start-programming.net/course/" target="_blank"><p>学習コース</p></a>
    </div>
</div>

②次は、cssでボタンの見た目を整えていきましょう。

.button{
    margin-top: 50px;
    display: flex;
}

.stapro,.course{
    /* ↓このwidth（ボタンの横幅）を決める方法が大事です！ */
    width: 25%;
    height:100px;
    background-color: rgb(187, 189, 192);
    text-align: center;
    margin: 0 auto;
}

.stapro:hover,.course:hover{
    background-color: rgb(147, 230, 147);
}

.stapro p,.course p{
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100%;
    margin: 0;
    font-size: 20px;
}

.stapro a,.course a{
    text-decoration: none;
    color: black;
}

ここでポイントとなるのが、ボタンの横幅を指定する方法についてです。
例えば、ボタンに対して「width="200px"」ように絶対値で指定しまうと、画面サイズが変わってもずっと200pxのままになってしまいます。

そのため、画面サイズに合わせてボタンのサイズが柔軟に変わるようにするために、「width=〇〇%」のように相対値で指定してあげます。
そうすることで、画面サイズを基準にボタンの大きさが変更されるようになります。
今回は「width=25%」と指定をしてあげましょう。

③現在のプログラムだと、ボタンの横幅は画面サイズに合わせて変わりますが、文字の大きさが変わらず、スマホサイズになった際に縦並びになりません。

これを直すために、cssを設定していきましょう。
まずは、480px以上960px以下（スマホより大きい画面からタブレットの画面まで）で見た時の設定を行います。

@media screen and (min-width: 480px) and (max-width: 960px) {
    .stapro p,.course p{
        font-size: 15px;
    }
}

これは、画面サイズ480pxから960pxまで、中に書いてあるcssを適用させるという意味になります。
適用させたいcssの書き方は、普段cssで書いている記述方法と同じです。
今回は、画面サイズが小さくなると文字のサイズも小さくしたいので、font-size: 15px;というプログラムを書きます。

④最後に、画面幅が480px以下（スマホ対応）になるとボタンが縦に並び、文字のサイズもさらに小さくなるように設定していきましょう。
まずは、480px以下になった際にボタンを縦並びにしたいので、以下のようなプログラムを記述します。

@media screen and (max-width: 480px) { 
    /*　画面サイズ 480px以下に適用　*/
    .button{
        display: block;
    }
}

このプログラムでは、ボタンが 「display: flex;」で横並びになっているのを解除するために、「display: block;」を指定しています。
これにより、横並びが解除され、ボタンは縦並びなります。

このままではボタンがくっついているように見えるので、ボタン同士の感覚を開け位置を調整していきます。
そして、スマホサイズになった時に文字がさらに小さくなるよう、文字のサイズも調整します。
今回は、480px以下になったら文字サイズは12pxに設定しましょう。

@media screen and (max-width: 480px) { 
    /*　画面サイズ 480px以下に適用　*/
    .button{
        display: block;
    }

    .stapro,.course{
        margin: 20px auto;
        width:50%;
    }

    .stapro p,.course p{
        font-size: 12px;
    }
}

これで見本と同じプログラムが完成しました！
