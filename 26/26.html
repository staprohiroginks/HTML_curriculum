26.画像とテキストのレスポンシブ対応

画面サイズに合わせて画像やテキストの大きさが変わる方法を学習しましょう！

見本はこちら

①まずはHTMLで画像とテキストの準備をします。

<div class="box">
    <div class="dog_box">
        <p>テキストテキストテキストテキストテキスト<br>
            テキストテキストテキストテキストテキスト<br>
            テキストテキストテキストテキストテキスト</p>
        <img src="images/dog.jpg" alt="犬の画像">
    </div>
</div>


<div class="box">
    <div class="dog_box">
        <p>テキストテキストテキストテキストテキスト<br>
            テキストテキストテキストテキストテキスト<br>
            テキストテキストテキストテキストテキスト</p>
        <img src="images/dog.jpg" alt="犬の画像">
    </div>
</div>

②次にcssで見た目を整えていきます。

.box{
    width: 50%;
    margin: 0 auto;
    border: 1px solid black;
}

.dog_box{
    display: flex;
    justify-content: space-around;
}

.dog_box p{
    margin: auto 0;
    font-size: 1vw;
}

.dog_box img{
    width: 30%;
    padding: 10px 0;
}

全体を囲っている「.box」はボーダーで囲い、「width: 50%;」というパーセントで指定をしましょう。

画像とテキストが入っている「.dog_box」には、横並びにするための「display:flex;」を指定し、
要素を均等感覚に並べるための「justify-content:space-around;」を指定します。

pタグの「margin: auto 0;」を使用することで、テキストの上下の高さを自動で調整してくれています。
（autoは自動で中央になるように調整してくれます）

pタグの文字サイズの「font-size:1vw;」の意味ですが、vwは"viewport width"（ビューポートの幅）を表し、ブラウザの横幅を示しています。

たとえば、ビューポートの幅が1000pxの場合、font-size: 1vwを指定していると10pxのフォントサイズになります。（1000pxの1%）
ビューポートの幅が変化すると、フォントサイズもそれに応じて自動的に調整されます。

③次は、960px以下（タブレット以下のサイズ）で見た時の設定（レスポンシブ対応）を行いましょう。

@media screen and (max-width: 960px) { 
    /* 画面サイズ 960px以下に適用 */
    .dog_box {
        flex-direction: column;
        align-items: center;
      }

      .dog_box p{
        font-size: 2vw;
        margin-top: 10px;
    }

    .dog_box img{
        width: 80%;
    }
}

「.dog_box」に対して「flex-direction:column;」とすることで、アイテムが垂直方向に配置されるように変更できます。
これをすることで、横並びだったアイテムが縦方向に並ぶようになります。

画面が小さくなるにつれて文字が見えにくくなるのを防ぐために、「font-size:2vw;」とし、画面幅に対して少し文字が大きくなるように設定しています。

④最後に、480px以下（スマホサイズ）で見た時の設定（レスポンシブ対応）を行いましょう。

@media screen and (max-width: 480px) { 
    /* 画面サイズ 480px以下に適用 */
      .dog_box p{
        font-size: 3vw;
    }
}

スマホだと文字が小さく見えるので、「 font-size: 3vw;」とすることで文字を大きく見せています。

これで、画面サイズに合わせて画像やテキストの大きさが変わるプログラムが完成しました！
