画像の大きさの変更

前回に引き続き、CSSファイルを活用して画像の大きさを変更しましょう！

【復習】画像の表示

まずは03. 画像と埋め込みを復習しながら、ネコの画像を表示しましょう！

index.html

<body>
    <img src="cat.jpg" alt="cat">
</body>

実行例

横幅の指定方法

画像の横幅の大きさを指定しましょう。
セレクタは「img」を指定します。
プロパティは「幅」の英語である「width(ウィズ)」が入ります。

style.css

img {
    width: 300px;
}

実行例

高さの指定方法

次は画像の高さを指定しましょう。
プロパティは「高さ」の英語である「height(ハイト)」が入ります。

style.css

img {
    width: 300px;
    height: 300px;
}

実行例

このように、横幅と高さを両方とも指定すると、画像の比率が変化します。

タブの大きさに合わせた指定方法

今までは画像の横幅や高さをピクセル（px）で設定してきました。
もしタブの大きさに合わせて画像の大きさを変化させたいなら、パーセント（%）で値を指定します。
試しにタブの横幅いっぱいに画像を表示しましょう。

style.css

img {
    width: 100%;
}

実行例

これで、タブの大きさを変更したら、画像の大きさも変化するようになりました！
