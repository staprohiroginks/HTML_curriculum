hover

hoverは「マウスをかざしたとき」を指定する疑似クラスの一つです。
疑似クラスと似たもので疑似要素というものもあります。

"疑似（ぎじ）"とは？

ここでいう疑似とは「実際のHTMLコードには存在しない仮想的なもの」という意味合いで使われます。
疑似クラス：マウスをかざしたときやクリックしたときなど、特定の動作が行われている状態のみ適用するセレクタ
疑似要素 ：HTMLコードで指定していない範囲を変更したり、存在していない要素を挿入したりするセレクタ

疑似クラス：hover

hoverは「マウスをかざしたとき」に表示方法を変化させる疑似クラスです。
まずは以下のコードを用意してください。

index.html

<h1>四季</h1>
<p>
    春・夏・秋・冬で構成される4つの季節を四季といいます。
    <br>日本をはじめとする、多くの国々で存在している自然現象です。
</p>
<div class="content">
  <h2>春</h2>
  <p>
      日本では3~5月頃の暖かい時期を指します。
      <br>桜の開花は春の象徴です。
  </p>
  <h2>夏</h2>
  <p>
      日本では6~8月頃の暑い時期を指します。
      <br>梅雨が明けるとプール日和が続きます。
  </p>
  <h2>秋</h2>
  <p>
      日本では9~11月頃の涼しい時期を指します。
      <br><a href="https://www.pref.hiroshima.lg.jp/" target="new">広島県</a>でも紅葉狩りが楽しめます。
  </p>
  <h2>冬</h2>
  <p>
      日本では12~2月頃の寒い時期を指します。
      <br><a href="https://en.wikipedia.org/wiki/Christmas" target="new">クリスマス</a>や年末年始で街がにぎわいます。
  <p>
</div>

実行結果

ここで「四季」という文字にマウスをかざしたら、文字の色が薄くなるようにしましょう。
cssファイルで以下のコードを書いてください。

style.css

h1 {
    padding: 5px;
}
h1:hover {
    opacity: 0.5;
}

実行結果

マウスをかざすと文字の色が薄くなりました！
opacityは「透明度」を表すプロパティであり、
カリキュラム13. 文字の配置（text-align：centerなど）で復習できます。

疑似クラス

cssファイルでは「.クラス名:疑似クラス名{}」で指定します。
疑似クラスの値は、以下が挙げられます。

疑似クラス名
.クラス名:〇〇{}		 効果が適用される要素
hover マウスがホバーしたとき
link 未訪問のリンク
visited 訪問済のリンク
active アクティブ状態になったとき
nth-child(n) 親要素内のn番目
nth-of-type(n) 親要素内の同じ種類の要素の中のn番目
not(B) B以外の要素

これらを利用して表示を変化させます。
先ほどのcssファイルに以下のコードを追加してみましょう。

style.css

h2 {
    width: 80px;
}
.content p {
    width: 350px;
}
:nth-child(6) {
    background-color: orange;
}
h2:nth-of-type(4) {
    color: white;
    background-color: black;
}

実行結果

すべての要素の中で上から6つ目の背景色がオレンジ色になりました！
さらにh2タグの4番目の背景色が黒色になりました！

疑似要素

cssファイルでは「.クラス名::疑似要素名{}」で指定します。
疑似クラスと違い、コロンが2つ必要です。
さらに細かいポイントのみ効果を付けることができます。
疑似要素の値は、以下が挙げられます。

疑似要素名
.クラス名::〇〇{}		 効果が適用される要素
first-letter 要素の一文字目
first-line 要素の一行目
before 要素の直前にコンテンツを追加
after 要素の直後にコンテンツを追加
backdrop 全画面表示の余白部分
marker 箇条書きの点や数字
placeholder inputやtextareaのプレイスホルダー
selection 選択した文書の一部

先ほどのcssファイルに以下のコードを追加してみましょう。

style.css

p::first-letter {
    color: red;
}
h2::before {
    content: "★";
    color: orange;
}

実行結果

pタグの最初の文字が赤くなりました！
さらにh2タグの最初に★マークが付きました！

練習問題

カリキュラム13. 文字の配置（text-align：centerなど）の復習もかねて、先ほどのコードに以下の3つの効果を加えましょう。
（見本はこちら）

  1. 【復習】春夏秋冬の内容（pタグで囲まれている部分）を全て中央揃えにする。
  2. 上から7行目の「夏」という文字にマウスを乗せると（ホバーすると）文字が赤色に変化する。
    お手本：夏
  3. 下から4行目の「広島県」のリンクをクリックした後、文字が緑色に変化する。
    お手本：クリスマス
